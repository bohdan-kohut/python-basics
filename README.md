# 🚀 Quotes Manager PRO

> CLI application for scraping, storing, and managing quotes using Python.

---

## 📌 About Project

**Quotes Manager PRO** — це консольний додаток, який демонструє повний цикл роботи з даними:

- отримання даних з сайту (web scraping)
- обробка HTML
- збереження в базу даних
- пошук, редагування та видалення інформації

Це мій практичний проект для прокачки навичок Python та підготовки до роботи.
